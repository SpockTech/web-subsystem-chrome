# Web Susbsystem for Chrome

Web Subsystem for Chrome or WSC is an operating system simulation that runs entirely in your web browser. The OS has persistent storage, a BIOS, and a recovery menu. It has 32 MB storage limit excluding the system files, simulates 1 GB of RAM, and 2 CPU cores. You can run this on any modern web browse

## Download
You can download the official HTML files for the OS here.

**Download steps:**
1. Click the `<> Code` button
2. Once you clicked the `<> Code` button, click the `Download ZIP` button. If you prefer the comand line, you can use the code `gh repo clone SpockTech/web-subsystem-chrome`
3. Extract the files to a folder
4. Run `index.html` and enjoy!
