# Web Susbsystem for Chrome

Web Subsystem for Chrome or WSC is an operating system simulation that runs entirely in your web browser. The OS has persistent storage, a BIOS, and a recovery menu. It has 32 MB storage limit excluding the system files, simulates 1 GB of RAM, and 2 CPU cores.
